Notes for all

ABOUT


Our project is contingent on productivity. It is an all-in-one workplace where you can write, plan, and get organized. This extension will make your work easier by accessing it on the same page. This extension will allow you to clip all or parts of web pages. It is an application that provides components such as notes, calendars, a Digital clock, background music, a to-do list, shortcut icons, and reminders.Being able to do all these things using only one app is a dream come true. It has an awesome feature where you can block other things that distract you during your productivity period.

Inspiration:

In this current scenario, students have been facing many issues regarding managing their notes and being productive throughout the day. SO, we came up with an idea of making a productivity extension this can be integrated into the curriculum and impact the learning of all students. Many different features can help them out.
